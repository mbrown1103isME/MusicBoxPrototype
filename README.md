# MusicBox Prototype

## The Goal:
- Create a streamlined music reviewing media app
- Function as a hub for discovering new music
- Spark discourse around famous albums and or bands

## The Stack:
- MERN stack
- MONGODB, Express, and Node for backend 
- React for the frontend

## Functionality
- Login and Register under a MusicBox account
- See other community reviews of music albums and like and comment
- View users profiles with their recently reviewed albums
- Search the Spotify API by Album, Artist, or search for users in MusicBox

## Reproducability
1. Start two terminal windows
2. Start client
    1. `cd client/my-app` 
    2. `npm install`
    3. `npm start `
3. Start server
    1. `cd server `
    2. `npm install`
    3. `npm run dev`
