# MusicBox Prototype

## The Goal:
- Create a streamlined music reviewing media app
- Function as a hub for discovering new music
- Spark discourse around famous albums and or bands

## The Stack:
- MERN stack
- MONGODB, Express, and Node for backend 
- React for the frontend

## Reproducability
- Start client
1. `cd client/my-app`
2. `npm install`
3. `npm start `
- Start server
1. `cd server `
2. `npm install`
3. `npm run dev`
